# 1.簡介
使用Framework "XPlan"與AR Foundation建立的XPlan_AR，主要演示XPlan搭配AR Foundation的應用。
 
# 2.安裝指南
### 系統要求
- [XPlan 2.0.2](https://github.com/asail0712/XPlanDemo)
- AR Foundation 5.1.5
### 安裝指南
使用Unity開啟這個項目。
需要配備攝影機。
